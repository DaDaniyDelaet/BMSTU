# Лабораторная работа №1

## Тема

Распечатка GIMPLE/IR из компилятора GCC с помощью GCC-плагина.

## Цель работы

Изучить промежуточное представление GIMPLE/IR в компиляторе GCC и реализовать плагин, который выводит информацию о структуре программы во время компиляции.

## Описание

В работе реализован GCC-плагин на C++, который подключается к компилятору GCC с помощью флага `-fplugin`.

Плагин запускается после построения SSA-представления программы и выполняет обход GIMPLE-представления функции. Для каждой функции выводятся базовые блоки, их предшественники, последователи, PHI-функции и GIMPLE-инструкции.

Для вывода используется `printf`.
