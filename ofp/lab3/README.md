# Лабораторная работа №3

## Тема
Разработка обобщённых классов на языке Scala с использованием неявных преобразований типов

## Цель

Приобретение навыков разработки обобщённых классов и использования неявных параметров.

## Вариант

Класс:

```scala
Polynom[T]
```

Реализованы:

- вычисление степени полинома
- дифференцирование
- использование generic типов
- использование implicit параметров

## Используемые конструкции

- Generic-классы `[T]`
- `implicit`
- `Numeric[T]`
- `match/case`
- рекурсия
