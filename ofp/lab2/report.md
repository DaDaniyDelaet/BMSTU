# Цель работы

Изучение базовых объектно-ориентированных возможностей языка Scala.

# Индивидуальный вариант

Последовательность, конструируемая как арифметическая прогрессия:

a(n)=a0+dn

Сумма двух последовательностей:

a=a1+a2 ⟺ ∀n≥0: a(n)=a1(n)+a2(n)

Умножение последовательности на число:

a=k*a1 ⟺ ∀n≥0: a(n)=ka1(n)

Обнуление всех членов последовательности, начиная с заданного номера:

a(n)=0, n≥k

Также требуется реализовать метод get, возвращающий i-й элемент последовательности.

# Реализация

В ходе выполнения лабораторной работы был разработан класс Sequence,
представляющий арифметическую прогрессию. Объекты класса являются неизменяемыми,
так как параметры объявлены с использованием ключевого слова val.

Класс содержит:

- поле a0 — первый элемент последовательности;
- поле d — разность арифметической прогрессии;
- поле limit — индекс начала обнуления элементов;
- метод get для вычисления элемента последовательности;
- перегруженные операторы +, * и /.

В ходе разработки использовались следующие возможности языка Scala:

- создание классов;
- конструкторы;
- методы классов;
- перегрузка операторов;
- создание новых объектов;
- условные конструкции if/else;
- неизменяемые значения val.

Реализация класса:

```scala
class Sequence(val a0: Int, val d: Int, val limit: Int = -1) {

  def get(i: Int): Int =
    if (limit != -1 && i >= limit) 0
    else a0 + d * i

  def +(other: Sequence): Sequence =
    new Sequence(a0 + other.a0, d + other.d)

  def *(k: Int): Sequence =
    new Sequence(a0 * k, d * k, limit)

  def /(n: Int): Sequence =
    new Sequence(a0, d, n)

  override def toString: String =
    "Sequence(" + a0 + ", " + d + ")"
}
```

Метод get вычисляет i-й элемент по формуле арифметической прогрессии:

```text
a(n)=a0+d*n
```

Оператор + создаёт новую последовательность, параметры которой являются суммой
соответствующих параметров двух исходных последовательностей.

Оператор * выполняет умножение последовательности на число.

Оператор / задаёт границу, начиная с которой элементы последовательности становятся равными нулю.

# Тестирование

```scala
scala> val s1 = new Sequence(1,2)

s1: Sequence = Sequence(1, 2)

scala> val s2 = new Sequence(3,1)

s2: Sequence = Sequence(3, 1)

scala> s1.get(4)

res0: Int = 9

scala> (s1+s2).get(4)

res1: Int = 16

scala> (s1*3).get(4)

res2: Int = 27

scala> val s3=s1/3

s3: Sequence = Sequence(1, 2)

scala> s3.get(0)

res3: Int = 1

scala> s3.get(1)

res4: Int = 3

scala> s3.get(2)

res5: Int = 5

scala> s3.get(3)

res6: Int = 0

scala> s3.get(4)

res7: Int = 0
```
