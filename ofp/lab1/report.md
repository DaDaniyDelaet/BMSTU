# Цель работы

Изучение основ функционального программирования на языке Scala с использованием чистых функций.

# Индивидуальный вариант

Функция reverse: List[List[Int]] => List[List[Int]],
выполняющая переворачивание как самого списка списков, так и каждого из его элементов.

# Реализация

Для выполнения лабораторной работы была установлена среда выполнения языка Scala версии 2.13.18.
Для работы Scala также была установлена Java Development Kit (JDK), поскольку REPL-интерпретатор
 Scala выполняется поверх Java Virtual Machine (JVM).

В ходе подготовки среды были выполнены следующие действия:

1. Скачан архив Scala версии 2.13.18.
2. Архив распакован в каталог:

```text
D:\scala\scala-2.13.18
```

3. В системную переменную PATH добавлен путь:

```text
D:\scala\scala-2.13.18\bin
```

4. Установлена Java Development Kit (JDK).

5. Выполнена проверка корректности установки Java:

```cmd
java -version
```

6. Выполнен запуск REPL-интерпретатора:

```cmd
scala
```

После запуска появилась командная строка интерпретатора:

```scala
scala>
```

Для ввода многострочного кода использовался специальный режим:

```scala
scala> :paste
```

После завершения ввода использовалась комбинация клавиш Ctrl+D.

В лабораторной работе была реализована функция reverse,
которая выполняет переворачивание внешнего списка и каждого вложенного списка.
Реализация выполнена с использованием рекурсии и механизма сопоставления с образцом
(pattern matching). Использование встроенной функции reverse не применялось.

```scala
def reverse(lst: List[List[Int]]): List[List[Int]] = lst match {
  case Nil => Nil
  case x :: xs =>
    def revInner(l: List[Int]): List[Int] = l match {
      case Nil => Nil
      case h :: t => revInner(t) ::: List(h)
    }

    reverse(xs) ::: List(revInner(x))
}
```

В реализации использовались следующие элементы языка Scala:

- `match/case` — механизм сопоставления с образцом;
- `Nil` — обозначение пустого списка;
- `::` — разделение списка на первый элемент и хвост;
- `:::` — объединение списков;
- рекурсивный вызов функций.
