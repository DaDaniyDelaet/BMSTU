# Лабораторная работа №1

## Тема
Введение в функциональное программирование на языке Scala

## Цель
Изучение основ функционального программирования на языке Scala с использованием чистых функций.

## Вариант

Функция:

```scala
reverse: List[List[Int]] => List[List[Int]]
```

Функция выполняет переворачивание внешнего списка и каждого вложенного списка.

## Запуск

Открыть Scala REPL:

```bash
scala
```

Вставить код:

```scala
:paste
```

После вставки нажать:

```text
Ctrl+D
```

Пример:

```scala
reverse(List(List(1,2,3), List(4,5)))
```

Результат:

```scala
List(List(5,4), List(3,2,1))
```
